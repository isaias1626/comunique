## Dependecies
 
 * install requirements
 
 ```
 pip install -r requirements.txt
 ```

 * Install dependecies

 ```
python -m pip install pillow
```

## Install

 ```
> git clone https://github.com/PedroHenriqueDevBR/social-network.git
> cd social-network
> python manage.py makemigrations
> python manage.py migrate
 ```
